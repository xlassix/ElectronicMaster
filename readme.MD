## Step 
 * Install Pipenv

 ```
 pip install pipenv
 ```

 * Install Dependencies

 ```
 pipenv install
 ```

 * Initialize enviroment 

```
 pipenv shell
```

 * Set input and output Dir in Folder at `ElectronicMaster.py` and line 10 and 11 respectively
  
 * Run program

```
 python ElectronicMaster.py 
```
